# Feature Roadmap

## Current Version (v0.1.0)

### ✅ Implemented
- [x] Event creation and management
- [x] Section editing with reordering
- [x] Template system with 6 predefined templates
- [x] Live preview of invitations
- [x] SQLite database with Prisma ORM
- [x] REST API endpoints
- [x] Responsive design with Tailwind CSS
- [x] Dashboard for event listing

## Contribution Guidelines

Interested in contributing? Check out the issues labeled:
- `good first issue` - Great for newcomers
- `help wanted` - Areas needing assistance
- `enhancement` - Feature requests

## Priority Features

Based on user feedback, these are the most requested:

1. **PDF Export** - Users want to save and print invitations
2. **Drag & Drop** - More intuitive section reordering
3. **More Templates** - Additional design options
4. **Mobile Support** - Full mobile app experience

Let us know which feature you'd like to see next!
